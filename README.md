# Contact Manager – Aplicação CRUD Simples

Este projeto é um mini aplicativo CRUD (Create, Read, Update, Delete) desenvolvido para gerenciar contatos de forma simples e intuitiva.  
Foi criado com o objetivo de demonstrar conhecimentos em lógica de programação, manipulação de DOM, armazenamento local e organização de código em JavaScript.

Este projeto faz parte do meu portfólio e reforça minhas habilidades iniciais em desenvolvimento web para vagas de **Estágio em Desenvolvimento Fullstack**.

---

## Funcionalidades

- Adicionar novos contatos  
- Listar todos os contatos cadastrados  
- Editar informações existentes  
- Excluir contatos individualmente  
- Armazenamento automático via **LocalStorage**  
- Interface simples, leve e responsiva  
- Código organizado em funções reutilizáveis

---

## Tecnologias utilizadas

- **HTML5** – Estrutura da interface  
- **CSS3** – Estilização e layout  
- **JavaScript** – Lógica, validação e manipulação de DOM  
- **LocalStorage** – Persistência simples de dados no navegador
